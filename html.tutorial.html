HTML Tutorial

Що таке HTML?

HTML — це мова розмітки, яка використовується для створення веб-сторінок.

Основне поняття

HTML тег — це елемент, який використовується для структурування вмісту сторінки.

Приклад HTML коду


<html>
   <head>
      <title>My page</title>
   </head>

   <body>
      <h1>Hello World</h1>
   </body>
</html>
Форматування тексту

Текст може мати різний колір та жирний стиль.

Висновок

HTML є основою для створення будь-якої веб-сторінки.
